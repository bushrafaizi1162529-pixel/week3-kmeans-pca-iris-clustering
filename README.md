# week3-kmeans-pca-iris-clustering

## Objective
Perform K-Means Clustering and PCA on the Iris Dataset.

## Tasks Performed
- Data Preprocessing
- Feature Scaling
- Elbow Method
- K-Means Clustering
- PCA (Dimensionality Reduction)
- Cluster Visualization
- Model Saving and Loading

## Dataset
Iris Dataset

## Libraries Used
- pandas
- numpy
- matplotlib
- scikit-learn
- joblib

## Results
- Applied K-Means with k=3
- Reduced dimensions using PCA
- Visualized clusters
- Saved trained model
